## 2.0版本更新：
- 增加了日语和中文两种语言选项。
- 增加了/wsde_list命令，列出所有可用的语音。

## 功能
/wsde (语音名字) (jp/cn 默认日文)   #随机维什戴尔游戏内语音。
/wsde_list    #列出所有可用的语音。
